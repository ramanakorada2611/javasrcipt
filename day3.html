Ans:The handling of this is also different in arrow functions compared to regular functions. In short, with arrow functions there are no binding of this. With arrow functions the this keyword always represents the object that defined the arrow function.

4.What is a Temporal Dead Zone?

Ans:If you are accessing any variable before delcaring with the let and const keywords we will get"Cannot access 'b' before initialization"because the variables is in temporal dead zone. To avoid TDZ ,try to access variable after its declaration and initialization. 

Example:console.log(c)
        var c=14//undefined
        
        console.log(b)
        let b=13//cannot access before initialization
